# Initiation à TDD (Test-Driven Development) avec TypeScript

Système factice de météo.

## Installation

Avant toute utilisation, installer les dépendances NPM :

`npm install`

## Exécution

`npm start`

## Développement avec Hot Reloading

`npm run watch`

## Build pour la production

`npm run build`
